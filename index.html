Schreiben B2 - Musterprüfung

Bearbeiten Sie die folgenden Aufgaben. Schreiben Sie jeweils ca. 120–180 Wörter.

Ihre Angaben





Thema 1: Geld

Aufgabe: Schreiben Sie einen formellen Brief an eine Bank oder Beratungsstelle, um Ratschläge zur besseren Geldverwaltung zu erhalten.

Thema 2: Filme

Aufgabe: Schreiben Sie einen argumentativen Text für eine Schülerzeitung oder Website. Diskutieren Sie: „Sind Filme nur Unterhaltung, oder können sie auch Bildung vermitteln?“

Thema 3: Musik

Aufgabe: Schreiben Sie eine E-Mail an eine Musikschule oder einen Konzertveranstalter, um sich über einen Kurs oder Workshop zu informieren.
